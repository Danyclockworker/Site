Quick Shutdown Shortcut30.04.2022

There's a little-known way to create a shortcut that will perform shutdown actions instantly (for lazy people like me).
Create a shortcut and enter the following in the Target field:
C:\Windows\System32\rundll32.exe PowrProf.dll,SetSuspendState, this is for hibernation.
For shutting down the system:
C:\Windows\System32\rundll32.exe PowrProf.dll, shutdown -s -t 00
You can download it from this link


p.s. If you have a different Windows location, don't forget to modify it accordingly.
