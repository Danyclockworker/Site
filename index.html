Ярлык для быстрого выключения30.04.2022

Есть малоизвестный способ создать ярлык который будет это делать моментально(для ленивых как я).
Создаем ярлык и прописываем в разделе Объект:
C:\Windows\System32\rundll32.exe PowrProf.dll,SetSuspendState, это для гибернации.
Для завершения работы
C:\Windows\System32\rundll32.exe PowrProf.dll, shutdown -s -t 00
Скачать его можно по этой ссылке


p.s. Если у вас другое местоположение Windows, его нужно не забыть изменить.
